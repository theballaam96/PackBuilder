# DK64 Pack Builder

PackBuilder is a web-based tool designed to build music and asset packs for Donkey Kong 64 ROM hacking and modding projects. This is served online at [https://theballaam96.github.io/PackBuilder/](https://theballaam96.github.io/PackBuilder/).

## Features

- Generate organized asset packs for DK64 modding
- Listen to songs created by the community
- Output generated can be used in other DK64 Hacking Projects

## Requirements

PackBuilder runs as a client-side JavaScript/HTML app (served from the repo).

## Quick Start

Clone the repository:

```
git clone https://github.com/theballaam96/PackBuilder.git
```

Open the builder in your browser:

```
python -m http.server
```
